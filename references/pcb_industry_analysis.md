# PCB行业岗位匹配实战案例

## 案例背景
- **候选人**：李三
- **学历**：南京邮电大学 信息工程（本科）
- **工作经历**：芯片测试工程师（约1年）
- **目标行业**：PCB（印刷电路板）

## 候选人技能清单
- 车规级IGBT、MOSFET可靠性认证
- AEC-Q101标准
- 环境试验箱、温湿度试验箱操作
- HTRB、H3TRB、温度循环等寿命试验
- 万用表、示波器操作
- 电性能测试参数分析
- 检测报告撰写

## 匹配分析结果

### PCB测试工程师：80分（高度匹配）
| 维度 | 得分 | 理由 |
|------|------|------|
| 技能 | 85 | 测试设备操作经验可迁移 |
| 经验 | 70 | 芯片测试→PCB测试有共通性 |
| 教育 | 80 | 信息工程专业对口 |
| 文化 | 75 | 注重细节、团队协作 |
| 潜力 | 85 | 学习能力强 |

### 可靠性测试工程师：75分
- 核心优势：有可靠性测试方法论
- 可迁移：AEC标准经验适用于汽车电子PCB

### 品质工程师：70分
- 有质量检测意识
- 需补充：IPC标准、PCB制造工艺

### PCB设计工程师：55分
- 有电子基础（模电、数电）
- 缺少：Altium Designer、Cadence等工具
- 建议：3-6个月学习后再投

## 关键洞察

### 可迁移技能识别
1. **测试方法论**：芯片可靠性测试方法同样适用于PCB
2. **设备操作**：示波器、万用表等通用仪器
3. **标准理解**：AEC-Q101可迁移到汽车电子PCB领域
4. **报告能力**：检测报告撰写是通用能力

### 行业关键词映射
| 芯片测试领域 | PCB领域 |
|-------------|---------|
| IGBT/MOSFET | PCB板/电路板 |
| AEC-Q101 | IPC标准 |
| HTRB/H3TRB | 热循环测试 |
| 可靠性认证 | 可靠性测试 |
| 失效分析(FA) | 失效分析 |

## 简历优化建议（针对PCB岗位）
1. 添加"PCB"、"电路板"、"SMT"等关键词
2. 强调测试方法论的通用性
3. 突出汽车电子相关经验（车规级产品）
4. 量化成果（通过率、管理测试单元数等）
